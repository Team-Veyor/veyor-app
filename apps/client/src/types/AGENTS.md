# types/ — 프론트 전용 공통 타입

프론트엔드 내부에서 여러 컴포넌트나 화면이 공유하는 타입이 모이는 곳입니다.
client/server가 함께 써야 하는 타입은 이곳이 아니라 `packages/shared`에 둡니다.

---

## 현재 구성

| 파일 | 역할 |
| --- | --- |
| `types.ts` | 공통 `Size` union 타입 |

---

## 무엇이 들어가는가

- 프론트 UI 전용 공통 타입
- 여러 컴포넌트가 공유하는 variant/size 계열 타입
- 화면·컴포넌트 경계를 넘나드는 client-only 타입

---

## 규칙

- 컴포넌트 하나에서만 쓰는 props 타입은 해당 컴포넌트 파일 안에 둡니다.
- API 요청/응답처럼 server와 공유해야 하는 타입은 `packages/shared`로 이동합니다.
- 너무 넓은 전역 타입보다 실제 재사용되는 작은 union/type을 우선합니다.

# Color

Veyor 클라이언트에서 사용하는 색상 토큰입니다.

토큰 소스는 `tokens/global.json`(core) · `tokens/semantic.json`(semantic) 이며, `pnpm tokens:build` 결과인 `src/styles/tokens.generated.css` 의 CSS 변수를 참조합니다. 피그마(Tokens Studio)에서 값을 수정해 푸시 → 빌드만 하면 이 문서도 자동으로 반영됩니다.

## Semantic

의미 기반 토큰입니다. 컴포넌트에서는 가능한 한 이 토큰을 사용합니다.

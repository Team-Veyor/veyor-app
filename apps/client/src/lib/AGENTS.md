# lib/ — 프론트 공통 유틸

도메인에 종속되지 않는 프론트 전용 순수 유틸이 모이는 곳입니다.
현재는 Tailwind class 합성을 위한 `cn()` 유틸을 제공합니다.

---

## 현재 구성

| 파일 | 역할 |
| --- | --- |
| `utils.ts` | `clsx` + `tailwind-merge` 기반 `cn(...inputs)` class 합성 유틸 |

---

## 무엇이 들어가는가

- UI와 화면에서 공통으로 쓰는 순수 함수
- 브라우저/React에 강하게 묶이지 않은 작은 헬퍼
- 특정 도메인 지식 없이 재사용 가능한 프론트 유틸

---

## 규칙

- 도메인 API 호출, 서버 상태 관리, 인증 토큰 처리 등은 여기에 두지 않습니다.
- 한 컴포넌트에서만 쓰는 헬퍼는 해당 컴포넌트 파일 가까이에 둡니다.
- 공통 유틸로 승격할 때는 최소 2곳 이상에서 실제 재사용되는지 확인합니다.
- Tailwind class 합성은 `cn(base, variant, size, className)` 순서를 우선합니다.
